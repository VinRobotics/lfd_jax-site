lfd_jax: Learning from Demonstration Algorithms in JAX

Introduction API docs GitHub

A modular and extensible library bringing state-of-the-art Learning from Demonstration algorithms to real-world robotics with JAX.

lfd_jax is a software package developed by VinRobotics that provides implementations of common Learning from Demonstration (LfD) algorithms in JAX for robotic applications. The package is designed to be modular and extensible, making it easy to develop, test, and integrate new learning algorithms into robotic systems. For API documentation, please visit the documentation.

Getting Started

Create a virtual environment

                
  # Install uv if you haven't already
  curl -LsSf https://astral.sh/uv/install.sh | sh

  # Create and activate virtual environment with Python 3.10
  uv venv lfd_jax_env --python=3.10
  source lfd_jax_env/bin/activate

Install the package

                
  git clone --recursive https://github.com/VinRobotics/lfd_jax.git

  cd LfD_Jax
  uv pip install -e .

  # Install lerobot in editable mode
  cd third_party/lerobot
  uv pip install -e ".[aloha, pusht]"

  # Apply the patch
  patch -p1 < ../../third_party_patches/lerobot.patch

Run some examples

                
  # Behavior cloning using LeRobot Push-T dataset
  cd ../../lfd_jax
  python scripts/train_bc_pusht_lerobot.py algo=bc_lerobot env=pushT_keypoint dataset_type=lerobot wandb.enable=true
  training.steps=200000 training.eval_freq=1000 training.log_freq=1000

  # Diffusion policy using LeRobot Push-T dataset
  python scripts/train_dp_pusht_lerobot.py algo=dp_lerobot env=pushT_keypoint dataset_type=lerobot wandb.enable=true
  training.steps=200000 training.eval_freq=1000 training.log_freq=1000
